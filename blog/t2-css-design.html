Box Model

Border?

Let's say there is a box, and there are more boxes in no particular order before border. Border separates the boxes from each other.

Margin?

Now, margins exists just right outside of borders (boxes). You can manipulate the size of gaps between boxes by editing margin's width and height.

Padding?

Padding is a space between the box and the content inside of the box.
