Recursion

A recursive function/method calls itself. In order for a recursive function/method to terminate, one needs a base case. A base case means that when the recursion meets the 'case', it won't call itself recursively. One thing to note is to that each recursion needs to get closer to the base case. 
        
          def fib(n)
            return n if (0..1).include? n
            fib(n-1) + fib(n-2) if n > 1
          end
Here in this fibonacci example, each time the method runs, the value of n decrements until the value of n becomes less than 1.
